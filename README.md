# Mentiq Labs Tech Website

Professional website for Mentiq Labs Tech showcasing mobile applications. Built with pure HTML, CSS, and JavaScript.

## Features

- **Pure HTML/CSS/JS**: No frameworks or build tools required
- **Fully Responsive**: Works on all devices and screen sizes
- **Privacy-First Design**: All data processed locally
- **Modern UI**: Clean, responsive design
- **Fast Loading**: Lightweight and optimized

## Project Structure

```
mentiq_website/
├── html-version/          # Pure HTML/CSS version
│   ├── index.html         # Main homepage
│   ├── privacy-policy.html # Privacy policy page
│   ├── styles.css         # All CSS styles
│   ├── script.js          # JavaScript for interactivity
│   └── README.md          # HTML version documentation
├── public/                # Static assets (images, etc.)
└── README.md              # This file
```

## Usage

### Option 1: Direct Browser
Simply open `html-version/index.html` in your web browser. No server or build process needed!

### Option 2: Web Server
Deploy the `html-version` folder to any web server:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
