Privacy Policy for Game Booster Pro: Ultra Mode

Effective Date: January 15, 2025
Last Updated: January 15, 2025

This Privacy Policy describes how Mentiq Labs Tech ("we," "our," or "us") collects, uses, and protects your information when you use our Game Booster Pro: Ultra Mode mobile application (the "App").

1. Information We Collect

1.1 No Personal Data Collection

Game Booster Pro: Ultra Mode does not collect any personal information, including but not limited to:

  • Personal identification information (name, email, phone number)
  • Device identifiers or unique device IDs
  • Location data
  • Usage analytics or behavioral data
  • Contact information or address book data
  • Photos, videos, or other media files

1.2 Local Data Only

The app only stores the following data locally on your device:

  • Game Profiles: Your custom game configurations (brightness, volume, notification settings)
  • App Preferences: Theme selection, auto-detection settings, restoration preferences
  • Performance Data: Real-time FPS, ping, RAM usage (displayed only, not stored)
  • System Settings: Temporary device settings applied during gaming sessions

2. How We Use Your Information

Since we do not collect personal information, there is no personal data to use. The local data stored on your device is used solely for:

  • Applying your custom game profiles when launching games
  • Restoring your device settings after gaming sessions
  • Displaying real-time performance metrics
  • Maintaining your app preferences and settings

3. Data Storage and Security

3.1 Local Storage

All data is stored locally on your device using:

  • Room Database: For structured data storage (game profiles, settings)
  • Encrypted Preferences: For secure storage of sensitive settings
  • Local File System: For temporary data and cache files

3.2 Data Security

We implement the following security measures:

  • All sensitive data is encrypted using Android's built-in encryption
  • Data is stored in the app's private directory, accessible only by the app
  • No data transmission occurs - everything remains on your device
  • Regular security updates and patches are provided

4. App Permissions

Game Booster Pro: Ultra Mode requests the following permissions to provide its functionality:

4.1 Required Permissions

  • Notification Access: To block notifications during gaming sessions
  • Usage Access: To detect when games are launched and monitor app usage
  • VPN Permission: To create local VPN for internet blocking functionality
  • System Settings: To modify brightness, volume, and other device settings

4.2 Optional Permissions

  • Overlay Permission: For floating FPS/ping monitors (can be disabled)
  • Device Admin: For advanced system optimization (requires user activation)

5. Third-Party Services

Game Booster Pro: Ultra Mode does not integrate with any third-party services that collect personal data. The app operates completely independently and does not use:

  • Analytics services (Google Analytics, Firebase Analytics, etc.)
  • Crash reporting services
  • Advertising networks
  • Social media integrations
  • Cloud storage services
  • External APIs or web services

6. Data Sharing and Disclosure

We do not share, sell, or disclose any data to third parties. This includes:

  • No data transmission to external servers
  • No sharing with advertisers or marketing companies
  • No data sharing with government agencies (unless legally required)
  • No sharing with business partners or affiliates

7. Your Rights and Choices

Since we don't collect personal data, most traditional privacy rights don't apply. However, you have the following rights regarding the app:

7.1 Data Control

  • Delete App Data: Uninstalling the app removes all local data
  • Reset Settings: You can reset all app settings to default
  • Revoke Permissions: You can revoke app permissions at any time
  • Clear Cache: You can clear app cache through device settings

8. Children's Privacy

Game Booster Pro: Ultra Mode is suitable for users of all ages. Since we don't collect any personal information, there are no special considerations for children's privacy.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
